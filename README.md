# First web application built during the Database &amp; Web Application course in UiT.

A tiny, simple web app with a nav bar that displays content. Includes beautiful cat pictures.
Topics: HTML formatting, CSS, Flask.

To run: Clone the repo, set up a virtual env (highly recommended!), run pip install -r requirements.txt, then python app.py. Enjoy!
Or just dive straight into the static folder if you came only for the cat pictures and want to skip the "hard part".
